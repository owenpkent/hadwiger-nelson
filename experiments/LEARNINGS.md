# LEARNINGS

Synthesis surface for cross-architecture findings on the Hadwiger-Nelson problem. Updated whenever an experiment lands a structural result.

Format: one entry per finding. Lead with the finding, then context.

## Template

### LXX. <Finding title>

**Architecture**: 1 / 2 / 3 / 4 (or cross)

**Experiment(s)**: `e1a_...`, etc.

**Finding**: one to three sentences stating the structural result.

**Why it matters**: how this informs the search direction.

**Wrong-approach status**: does this method pass the $\mathbb{Q}^2$ / $L^\infty$ / $\mathbb{R}^1$ controls?
